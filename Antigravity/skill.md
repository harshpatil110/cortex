# Cortex Developer & System Skill Profile (skill.md)

This matrix defines the complete skill sets, operational fluencies, and core engineering capabilities required by any developer or AI agent executing the codebase for **Cortex**.

---

## 1. Full-Stack Engineering Expertise

### Frontend (React & Tailwind CSS)
*   **Minimalist UI Composition:** Implementation of high-end editorial layouts utilizing crisp whitespace, strict asymmetric alignments, and thin border controls (`border-stone-200`) explicitly matching the Warm Editorial Minimalism mandate.
*   **Dynamic Canvas Rendering:** Competency in canvas or mathematical vector graphs (e.g., D3.js, React Flow, or primitive SVGs) to illustrate high-performance interactive network clusters for the Knowledge Graph Visualizer.
*   **Browser Extension Systems:** Expertise in building Manifest V3 extensions tailored for Chrome/Arc, utilizing background service workers, content script injections, context menus, and native active tab data capture pipelines.

### Backend (Python & FastAPI)
*   **Asynchronous Processing:** Proficient use of `asyncio` routines, non-blocking network requests, and `FastAPI.BackgroundTasks` to handle heavy ingest pipelines out-of-band without degrading API response loops.
*   **Data Validation:** Absolute enforcement of type-hints and strict Pydantic V2 schema validations for incoming payloads and multi-dimensional JSON outputs.

---

## 2. Multimodal AI & Processing Pipelines

### Data Extraction & Isolation
*   **Video Processing:** Utilizing `ffmpeg-python` wrappers to manipulate media files, run silent audio layer extractions, and isolate video frames down to precise timestamped capture indices.
*   **Computer Vision Processing:** Configuring frame sampling matrices using OpenCV (`cv2`) to dynamically extract sequential static images every 1.5 to 2.0 seconds for downstream OCR evaluations.

### NVIDIA NIM API Integration
*   **Speech-to-Text Pipeline:** Routing audio binary payloads through standard OpenAI-compatible streaming routes to extract zero-loss verbal transcripts.
*   **Vision-LLM OCR Processing:** Orchestration of optimized vision models to transcribe deep-nested visual structural items (terminal code calls, database connection variables, UI pathways).
*   **Structured Synthesizers:** Writing defensive systemic prompts requiring LLMs to consistently output deterministic JSON code blocks that parse clean attributes without markdown conversational noise.

---

## 3. Database Architecture & Vector Operations

### Relational Storage (Supabase/PostgreSQL)
*   **Schema Design:** Construction of performance-tuned tables utilizing relational key mappings, cascading delete triggers, and complex JSONB storage operations.
*   **Lexical Indexing Optimization:** Implementing PostgreSQL Generalized Inverted Indexes (`GIN`) combined with `to_tsvector('english', ...)` fields to allow real-time keyword string verification across vast code snippets and text summaries.
*   **Multi-Tenant Isolation:** Implementing strict security patterns where all write, mutation, or lookup queries execute inside isolated constraints bounded by the active session `user_id`.

### Vector Operations (ChromaDB)
*   **Vector Engine Configuration:** Configuring persistent ChromaDB instances running cosine distance calculations across 1024-dimensional embedding architectures.
*   **Metadata Filtering:** Implementing dynamic metadata mapping strategies to pre-filter high-dimensional similarity matches safely inside specific user isolation parameters.

---

## 4. Scraping, Extraction & Automation Protocols

*   **Scraping Bypass Engineering:** Orchestration of specialized automated download engines (`yt-dlp`) and browser controllers (Playwright) configured to process ephemeral web media configurations safely.
*   **Session Management & Rotation:** Implementation of self-healing network wrappers utilizing rotating residential proxies, randomized browser user-agent headers, and adaptive failure timeouts to reliably bypass aggressive web anti-scraping firewalls.

---

## 5. Architectural Paradigms & Algorithms

### Reciprocal Rank Fusion (RRF)
*   Ability to implement blending ranking routines that normalize distribution discrepancies between high-speed BM25 relational database text strings and dense math-based ChromaDB similarity rankings to output a singularly unified result matrix.

### Hierarchical Semantic Clustering
*   Implementation of real-time semantic categorization architectures. Using the text embedding vector coordinates of incoming assets, the engine must evaluate geometric cluster spacing to dynamically categorize and maintain asset placement within user **Plates** without manual directory setups.
